This is a built-in template for a GuicedEE Redis Module.
A Redis Module extends RedisModule and configures a Vert.x Redis client with connection pooling.
This class will be automatically registered as a service loader implementation.
Predefined variables:
${NAME}   Name of the class to create
${PACKAGE_NAME}   Package name
${REDIS_NAME}   Logical Redis name (used as @Named qualifier)
${REDIS_HOST}   Redis server hostname (default: localhost)
${REDIS_PORT}   Redis server port (default: 6379)
${REDIS_DATABASE}   Redis database number (default: 0)
${MAX_POOL_SIZE}   Maximum connection pool size (default: 6)
