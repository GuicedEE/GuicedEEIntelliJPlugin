This is a built-in template for a GuicedEE RabbitMQ Consumer.
A RabbitMQ Consumer is a class that implements QueueConsumer and processes messages from a RabbitMQ queue.
This class will be automatically registered as a service loader implementation.
Predefined variables:
${NAME}   Name of the class to create
${PACKAGE_NAME}   Package name
${CONNECTION_NAME}   RabbitMQ connection name
${QUEUE_NAME}   RabbitMQ queue name
