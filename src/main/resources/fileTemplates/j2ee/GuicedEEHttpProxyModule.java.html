This is a built-in template for a GuicedEE HTTP Proxy Module.
An HTTP Proxy Module extends ProxyModule and configures a Vert.x HttpProxy reverse proxy.
This class will be automatically registered as a service loader implementation.
Predefined variables:
${NAME}   Name of the class to create
${PACKAGE_NAME}   Package name
${PROXY_NAME}   Logical proxy name (used as @Named qualifier)
${PROXY_PORT}   Port the proxy listens on (default: 8080)
${ORIGIN_HOST}   Origin server hostname
${ORIGIN_PORT}   Origin server port (default: 7070)
