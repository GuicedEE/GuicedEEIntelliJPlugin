This is a built-in template for a GuicedEE Persistence Module.
A Persistence Module is a class that extends DatabaseModule and configures database connectivity.
This class will be automatically registered as a service loader implementation.
Predefined variables:
${NAME}   Name of the class to create
${PACKAGE_NAME}   Package name
${PERSISTENCE_UNIT}   Name of the persistence unit
${JNDI_NAME}   JNDI name for the datasource (optional, can be left empty)
